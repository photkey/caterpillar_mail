caterpillar\_mail
=================

caterpillar\_mail 简介
----------------------

caterpillar\_mail
主要为收发邮件、解析邮件做的进一步高层封装，使用起来极其方便，追求极简的编码方式

caterpillar\_mail 安装方式
--------------------------

.. code:: bash

    pip install caterpillar_mail
