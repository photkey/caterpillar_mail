# caterpillar_mail
### caterpillar_mail 简介
caterpillar_mail 主要为收发邮件、解析邮件做的进一步高层封装，使用起来及其方便，追求极简的编码方式
### caterpillar_mail 安装方式
```bash
pip install caterpillar_mail
```
